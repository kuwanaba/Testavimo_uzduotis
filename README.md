# A framed greeting

The program greets you with a message surrounded in a frame of asterisks after you enter your name.

## How to run

 - `git clone git@github.com:kuwanaba/Testavimo_uzduotis.git`
 - `cd Testavimo_uzduotis`
 - `g++ main.cpp -o main`
 - `/.main`
